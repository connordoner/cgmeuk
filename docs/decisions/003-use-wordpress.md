# ADR 3: Use WordPress

## Context

I've [decided to migrate my personal website from Google Sites][impetus] but I
haven't decided on what to migrate to yet. I naturally need to do this before I
can get on with migrating (even if the answer is simply static HTML!).

## Decision

I'm going to migrate to WordPress and will use it to build and run my personal
website.

I'm going to host it myself on AWS as I have other infrastructure already hosted
there and would like to learn more about its intricacies.

## Consequences

As explained in my [decision record on starting this project][impetus], I'll
have to work harder than previously to keep my personal website available to the
world and up-to-date with modern web standards. I'll manage this risk by using
my resilience experience from [my startup][startup] and making my website highly
available (unless Russia nukes my host!).

I'll also have to spend more on hosting compared to when I used Google
Sites—especially given my intentions around resilience. However, I'm in a better
financial position now than I have been previously, and am therefore willing and
able to spend more.
